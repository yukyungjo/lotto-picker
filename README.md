# 로또 AI 웹앱

앱 설치 기능 없이 GitHub Pages에서 쓰는 로또 AI 웹앱입니다.

기능:
- 완전 자동 5줄
- AI 분석 추천 5줄
- GitHub Actions로 매주 당첨번호 자동 업데이트
- 제외 번호 설정
- 내가 산 번호 저장
- 당첨 확인
- 최근 100회 저장 기록
- 다크모드

업로드 후:
1. 이 zip을 풀어서 모든 파일을 GitHub 저장소에 올립니다.
2. Settings → Pages에서 main / root를 선택합니다.
3. Actions 탭에서 Update lotto numbers를 한 번 수동 실행합니다.
